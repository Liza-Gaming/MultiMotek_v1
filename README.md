# מולטימותק 🍭
משחק לימודי שנוצר עבור המרכז למען חולי סוכרת בישראל, שמטרתו ללמד ולהמחיש את חשיבות איזון רמות הסוכר.

 
## פיצ'רים

- **כפתור "בדיקת סוכר" (Sugar Check):** לחיצה מציגה את רמת הסוכר הנוכחית ומעדכנת את ה־UI.
- **רמות סוכר דינמיות:** הרמה משתנה בזמן אמת ובהשפעת אירועים במשחק.
- **השפעות רמות הסוכר בדם:** אפקטים ויזואליים (למשל טשטוש מתחת לסף) והתאמות תנועת השחקן.
- **השפעות מזג האוויר:** מזג האוויר משפיע על קצב כניסת האינסולין
- **דירוג השחקן והתקדמות:** ניקוד/לבבות ומדדים לסיכום שלב לפי זמן באיזון ויעדים שהושגו.
